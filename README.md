# `[Violet Core]`
`temporarily froze the idea`

An experimental module for optimizing high-load parts of the `AzerothCore (WotLK)` game server using Rust. The project aims to replace "heavy" C++ formatters and utilities with safe and fast Rust counterparts via FFI.

Replacing C++ abstractions with direct Rust/C interactions allows for significant offload of the World Loop.
But it takes a lot of time to do it.
